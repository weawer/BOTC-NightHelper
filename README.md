# Blood on the Clocktower Night-Helper

Single-page storyteller helper app for managing Trouble Brewing night flow.

## Features

- Player seating list with name + role assignment
- Status toggles for Alive/Dead, Drunk, and Poisoned
- Guided night order for First Night and Other Nights
- Automatic skipping of roles not in play (or inactive due to death without dead ability)
- Droison warning banner during role resolution
- Reminders for Demon kill and Monk protection
- Dawn summary with automatic death application

## Development

Install dependencies (already done during scaffold):

```bash
npm install
```

Run dev server:

```bash
npm run dev
```

Build production bundle:

```bash
npm run build
```
